# GPU 스펙 가이드: WSL vs Colab

## 🎯 핵심 답변: 어느 정도 GPU면 충분한가?

### ✅ **최소 스펙 (사용 가능)**
- **VRAM: 4GB 이상**
- **예시**: GTX 1050 Ti (4GB), GTX 1650 (4GB), RTX 3050 (8GB)
- **사용 가능한 모델**: ResNet18, EfficientNet-B0, ViT-Small
- **배치 크기**: 8-16
- **학습 시간**: 느리지만 실용적 (에폭당 10-30분, 데이터셋 크기에 따라)

### ✅ **권장 스펙 (편하게 사용)**
- **VRAM: 6-8GB 이상**
- **예시**: GTX 1660 Ti (6GB), RTX 2060 (6GB), RTX 3060 (12GB), RTX 3070 (8GB)
- **사용 가능한 모델**: ResNet50, EfficientNet-B3, ViT-Small/Base
- **배치 크기**: 16-32
- **학습 시간**: 빠름 (에폭당 5-15분)

### ✅ **이상적 스펙 (Colab보다 편함)**
- **VRAM: 12GB 이상**
- **예시**: RTX 3060 (12GB), RTX 3080 (10GB), RTX 4070 (12GB), RTX 4080 (16GB)
- **사용 가능한 모델**: 모든 모델 자유롭게 사용
- **배치 크기**: 32-64
- **학습 시간**: 매우 빠름 (에폭당 2-10분)

## 📊 모델별 필요 VRAM (배치 크기 16 기준)

| 모델 | 최소 VRAM | 권장 VRAM | 배치 16시 실제 사용량 |
|------|----------|----------|---------------------|
| ResNet18 | 2GB | 4GB | ~1.5GB |
| EfficientNet-B0 | 2GB | 4GB | ~1.0GB |
| ResNet50 | 4GB | 6GB | ~2.5GB |
| EfficientNet-B3 | 4GB | 6GB | ~2.0GB |
| ViT-Small | 4GB | 6GB | ~2.0GB |
| ViT-Base | 6GB | 8GB | ~4.0GB |
| Swin-Base | 8GB | 12GB | ~5.0GB |

**참고**: 배치 크기를 줄이면 더 적은 VRAM으로도 사용 가능합니다.

## 🔍 실제 사용 시나리오

### 시나리오 1: 4GB GPU (GTX 1050 Ti, GTX 1650)
```
✅ 가능한 작업:
- ResNet18 학습 (배치 8-16)
- EfficientNet-B0 학습 (배치 16)
- ViT-Small 학습 (배치 8)

⚠️ 제한사항:
- ResNet50은 배치 4-8로만 가능 (느림)
- ViT-Base는 사용 어려움

💡 권장:
- 프로젝트 완료 가능하지만 느림
- Colab과 비슷한 속도 (T4보다 약간 느림)
- 하지만 인터넷 연결 불필요, 무제한 사용 가능
```

### 시나리오 2: 6GB GPU (GTX 1660 Ti, RTX 2060)
```
✅ 가능한 작업:
- ResNet50 학습 (배치 16)

💡 권장:
- 대부분의 프로젝트에 충분

- 로컬에서 편하게 사용 가능
```

### 시나리오 3: 8GB GPU (RTX 3060, RTX 3070)
```
✅ 가능한 작업:
- 모든 모델 자유롭게 사용
- 큰 배치 크기 가능 (32-64)
- 여러 실험 동시 진행 가능

💡 권장:
- Colab보다 편함 (세션 제한 없음)
- 빠른 학습 속도

```

## 💰 가성비 관점

### Colab 사용 시
- **비용**: 무료
- **제한**: 세션 12시간, GPU 할당 불안정
- **편의성**: 데이터 업로드 필요, 인터넷 연결 필요

### WSL 사용 시 (GPU 구매 기준)
- **RTX 3060 12GB**: 약 30-40만원 (중고)
- **장점**: 무제한 사용, 빠른 속도, 로컬 개발 편의성
- **ROI**: 프로젝트 2-3개 하면 Colab 시간 절약으로 충분히 가치 있음

## 🎯 실전 권장사항

### "Colab 대신 쓸만한가?" 기준

#### ✅ **충분함 (Colab 대신 사용 가능)**
- **6GB 이상 GPU**
- 예: RTX 2060, RTX 3060, RTX 3070
- 이유: Colab T4와 비슷하거나 더 빠름, 세션 제한 없음

#### ⚠️ **가능하지만 느림 (선택적 사용)**
- **4GB GPU**
- 예: GTX 1050 Ti, GTX 1650
- 이유: 느리지만 완료 가능, Colab이 더 빠를 수 있음

#### ❌ **부족함 (Colab 권장)**
- **4GB 미만 또는 CPU만**
- 이유: 너무 느려서 실용적이지 않음

## 📝 실용적인 조언

### 현재 프로젝트 (AI 이미지 탐지) 기준

**최소 요구사항:**
- GPU: 4GB VRAM 이상
- 모델: ResNet18 또는 EfficientNet-B0
- 배치 크기: 8-16
- 학습 시간: 10 에폭 기준 3-5시간

**권장 사양:**
- GPU: 6-8GB VRAM
- 모델: ResNet50 또는 ViT-Small
- 배치 크기: 16-32
- 학습 시간: 10 에폭 기준 1.5-3시간

**이상적 사양:**
- GPU: 12GB VRAM 이상
- 모델: 모든 모델 자유롭게
- 배치 크기: 32-64
- 학습 시간: 10 에폭 기준 1-2시간

## 🔧 최적화 팁 (제한된 GPU에서)

1. **Gradient Accumulation 사용**
   - 작은 배치로 큰 배치 효과
   - VRAM 절약

2. **Mixed Precision Training (FP16)**
   - 메모리 사용량 절반으로 감소
   - 속도 1.5-2배 향상

3. **모델 체크포인팅**
   - 중간 저장으로 안전성 확보

4. **데이터 로딩 최적화**
   - `num_workers` 조정
   - `pin_memory=True` 사용

## 결론

**6GB 이상 GPU가 있으면 Colab 대신 WSL 사용을 강력히 권장합니다!**

이유:
1. ✅ Colab과 비슷하거나 더 빠른 속도
2. ✅ 세션 제한 없음 (무제한 학습)
3. ✅ 인터넷 연결 불필요
4. ✅ 로컬 개발 편의성
5. ✅ 데이터 프라이버시

**4GB GPU도 가능하지만**, 학습 시간이 길어서 인내심이 필요합니다.

**현재 프로젝트 기준으로는 6GB GPU면 충분히 Colab 대신 사용 가능합니다!**
